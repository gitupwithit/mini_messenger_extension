Mini Messenger

Info

This is an extrememly simple 2-way comminicator. It is meant to work like a walkie-talkie, (though a message that hasn't been received yet will wait for the user to receive it). It will not save more than one message. If you send a message before a previous message has been received, the previous message will be overwritten. Your gmail address, partner's gmail address and last unreceived message that you sent is stored on the server. To remove your info from the server, , or uninstall the extension. Or both.
