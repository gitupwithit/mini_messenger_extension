Mini Messenger Info

  • Mini Messenger's use and transfer of information received from Google APIs will adhere to the Google API Services User Data Policy, including the Limited Use requirements.
  • Mini Messenger is inspired by walkie talkies.
  • It is a two-way communicator.
  • Mini Messenger only saves the last message on the server if your partner is offline.
  • Sending a blank message will unsend your last message if it hasn't been received yet.
